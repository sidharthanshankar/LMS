📚 RVCE Advanced Library Management System (LMS)
Transforming Campus Resource Management
This Library Management System (LMS) was developed and successfully showcased at the RVCE Project Expo 2026. The platform modernizes traditional library workflows through an intuitive, data-driven administrative dashboard and a robust circulation engine.
The system is built to handle significant datasets, having successfully imported and managed 460+ active user records (Students & Staff) with high-performance query logic and real-time inventory tracking.
🚀 Key Features
1. Advanced Admin Dashboard
Inventory Insights: Real-time tracking of unique titles, issued copies, and balance stock.
Member Management: Categorized tracking of student and staff registrations.
Financial Oversight: Automated pending fine calculations for overdue returns.
Visitor Logs: Daily visitor traffic monitoring to analyze library usage patterns.
2. Seamless Circulation Workflow
Digital Request System: Students can request books, which librarians can approve or reject through a central management interface.
Manual Issue Logic: Ability for librarians to manually override and issue books via ISBN/Account Number.
Transaction History: Comprehensive logging of active borrowings and past return history.
3. Scalable Data Management
Bulk Import: Support for importing hundreds of user records via CSV (verified with 460+ entries).
Preplanned Architecture: Developed with an emphasis on extensive database pre-planning to ensure data integrity and minimize runtime errors.
🛠️ The Tech Stack
Core Backend
Language: Python
Framework: Flask (High-performance web micro-framework)
Database: SQLite (Relational database for ACID compliance)
Authentication: Flask-Login (Role-based access control)
Frontend & UI
Templating: Jinja2
Styling: Bootstrap 5 (Modern, responsive grid system)
Icons: FontAwesome 6+
